## Rodando localmente

Iniciando Git no Projeto

```bash
  git init
```

Verificar o Status

```bash
  git status
```

Adicione os arquivos nao rastreados

```bash
  git add .  - adiciona todos os arquivos 
  git add nome_arquivo - adiciona arquivo especifico

```

Commitar(adicionar no repositorio local)

```bash
  git commit -m 
```
